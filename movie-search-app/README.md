# Movie Search App

A modern, Netflix-inspired movie search application built with React and TMDB API.

## Features

- Debounced search with auto-complete
- Movie categories (Popular, Top Rated, Now Playing)
- Advanced filters (genres, year range, rating)
- Responsive design
- Favorites with localStorage
- Search history
- Detailed movie information modal
- Pagination with Load More

## Tech Stack

- React 18
- TMDB API
- CSS3 (Grid, Flexbox, Animations)
- localStorage

## Installation

\`\`\`bash
npm install
npm run dev
\`\`\`

## Environment Variables

Create `.env` file:
\`\`\`
VITE_TMDB_API_KEY=your_api_key_here
\`\`\`
